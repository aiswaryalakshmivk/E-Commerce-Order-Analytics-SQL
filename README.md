# E-Commerce Order Analytics SQL

## Overview
This project focuses on e-commerce business analysis using SQL.

It includes SQL queries to analyze:
- Customer orders
- Product sales
- Order trends
- Customer purchase insights

## Files Included

- `customers.csv` – Customer details
- `products.csv` – Product information
- `orders.csv` – Order transaction data
- `order_analysis.sql` – Order-based SQL analysis
- `product_analysis.sql` – Product performance analysis
- `customer_insights.sql` – Customer purchase insights

## Skills Used
- SQL
- Joins
- Aggregate Functions
- GROUP BY
- ORDER BY
- Business Analytics Concepts

## Purpose
To practice SQL queries using e-commerce data in a realistic business scenario.
